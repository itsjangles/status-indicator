Always-on-top setup (hidden help)

  1. Open your browser extension store (Chrome Web Store / Edge Add-ons / Firefox Add-ons).
  2. Search for an extension like always on top or pin window.
  3. Install the extension and allow any requested window permissions.
  4. Pin the extension to your toolbar (optional) for quick access.
  5. Activate it on this tab/window so the status page stays above other windows.
